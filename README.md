- 其中，原程序中
```
batch_Y = self.YTrain[start:end,:]
```
处，在处理Boston房价数据集时会出现`TMI`的错误，原因是此时处理完的Y并非一维矩阵
为解决该错误，在`HelperClass.DataReader_1_2.NomalizeY`中，在处理数据时直接将Y转为一维矩阵，然后在`GetBatchTrainSamples`中将代码改为
```
batch_Y = self.YTrain[start:end]
```
